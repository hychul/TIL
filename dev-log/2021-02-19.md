# 맥 메모에서 취소선 단축키 설정하기

맥에서는 기본으로 설정된 단축키 이외에 사용자가 직접 정의한 단축키를 추가할 수 있다. [시스템 환경 설정 - 키보드 - 단축키 - 앱 단축키] 에서 아래와 같은 방식으로 메뉴를 직접 선택을 하거나 '취소선'의 이름으로 단축키를 추가하면 된다.
